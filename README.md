# Wave Reveal
![screenshot](image.png)

A Three.js illustration to view and record the wave reveal and fade effect of a vertex geometry of a model.

A live demo is available at [wave-reveal.arnv.dev](https://wave-reveal.arnv.dev).

The web app can be used to preview, and record a video of the animation, it also supports uploading to custom stl models for visualization.
